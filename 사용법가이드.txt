================================================================================
                          🥁 BEAT DRUM v2.0 사용법 가이드 🥁
================================================================================

Beat Drum은 웹 브라우저에서 실행되는 실시간 드럼 비트 생성기입니다.
16스텝 시퀀서를 통해 직관적으로 드럼 패턴을 만들고 편집할 수 있습니다.

🎮 1. 기본 조작법
================================================================================

▶️ 재생 제어
- [Play 버튼]: 드럼 패턴 재생 시작
- [Pause 버튼]: 재생 일시정지 (패턴 유지)
- [Stop 버튼]: 재생 정지 및 처음으로 돌아가기

🎛️ 고급 템포 조절
- BPM 노브: 마우스 드래그로 정밀한 BPM 조절 (60~200)
- +/- 버튼: 1 BPM씩 세밀 조정 (길게 누르면 연속 조정)
- BPM 값 클릭: 직접 숫자 입력으로 정확한 값 설정
- 실시간 시각화: BPM에 따른 색상 변화 및 비트 펄스
- 장르 표시: 현재 BPM에 맞는 음악 장르 자동 표시
- 재생 중에도 부드러운 템포 변경 가능

🎵 패턴 편집
- 각 드럼 트랙의 스텝(동그라미)을 클릭하여 비트 ON/OFF
- 활성화된 스텝은 색상으로 표시됨
- 재생 중에도 실시간으로 패턴 수정 가능

🔊 사운드 미리듣기
- 각 트랙 옆의 🔊 버튼을 클릭하여 드럼 사운드 확인

🎲 패턴 도구
- [Clear 버튼]: 모든 패턴 지우기
- [Random 버튼]: 자동으로 랜덤 드럼 패턴 생성

💾 저장/불러오기
- [저장 버튼]: 현재 패턴을 이름을 지어 저장 (최대 20개)
- [불러오기 버튼]: 저장된 패턴 목록에서 선택하여 로드
- [다운로드 버튼]: 현재 패턴을 JSON 파일로 다운로드

🎵 프리셋 슬롯 (1-5번)
- [클릭]: 해당 슬롯의 패턴 로드
- [Shift + 클릭]: 현재 패턴을 해당 슬롯에 저장
- [우클릭]: 해당 슬롯의 패턴 삭제
- 저장된 슬롯은 파란색으로 표시됨

================================================================================
⌨️ 2. 키보드 단축키
================================================================================

🎮 재생 제어
Space바         - 재생/일시정지 토글
Esc             - 재생 정지
Shift + C       - 패턴 완전 지우기
Shift + R       - 랜덤 패턴 생성

💾 저장/불러오기 제어
Ctrl + S        - 패턴 저장 (이름 입력)
Ctrl + O        - 패턴 불러오기 (목록에서 선택)
Ctrl + D        - 패턴 다운로드 (JSON 파일)

🎵 프리셋 빠른 접근
F1~F5           - 프리셋 슬롯 1~5 로드
Shift + F1~F5   - 프리셋 슬롯 1~5에 저장

🎵 BPM 조절 빠른 접근
↑/↓             - BPM ±1 조절
←/→             - BPM ±5 조절  
+/-             - BPM ±1 조절
Shift + ↑/↓     - BPM ±10 조절

🥁 드럼 사운드 재생 (숫자키 1-9)
1               - Kick (킥드럼)
2               - Snare (스네어)
3               - Hi-hat (하이햇)
4               - Tom (탐)
5               - Open Hat (오픈 하이햇)
6               - Crash (크래시)
7               - Ride (라이드)
8               - Clap (클랩)
9               - Cowbell (카우벨)

🎼 추가 드럼 사운드 (문자키)
Q               - Shaker (셰이커)
W               - Hi Tom (하이 탐)
E               - Lo Tom (로우 탐)

💡 팁: 키보드로 드럼 사운드를 재생하면서 리듬감을 익힐 수 있습니다!

� 4. 패턴 저장 & 관리 시스템
================================================================================

🗂️ 저장 방식 (3가지)

▶️ 1. 로컬 저장 (브라우저 내장)
- 최대 20개 패턴 저장 가능
- 패턴 이름, 생성 시간 자동 기록
- 브라우저를 닫아도 데이터 유지
- 사용법: [저장] 버튼 클릭 → 이름 입력

▶️ 2. 프리셋 슬롯 (빠른 접근)
- 5개의 빠른 접근 슬롯 (1~5번)
- F1~F5 키로 즉시 로드/저장
- 클릭/Shift+클릭으로도 조작 가능
- 저장된 슬롯은 파란색으로 표시

▶️ 3. 파일 저장 (JSON 형식)
- 컴퓨터에 파일로 백업/공유
- 다른 기기에서도 불러오기 가능
- JSON 형식으로 모든 패턴 데이터 포함
- 사용법: [다운로드] 버튼 → 파일 저장

📝 저장되는 정보
- 12개 드럼 트랙별 16스텝 패턴
- BPM 설정값
- 패턴 이름 및 생성 시간
- Beat Drum 버전 정보

🔄 불러오기 방법

▶️ 로컬 저장에서 불러오기:
1. [불러오기] 버튼 클릭
2. 저장된 패턴 목록 확인
3. 번호 입력하여 선택

▶️ 파일에서 불러오기:
1. [불러오기] 버튼 → '파일' 입력
2. 또는 직접 JSON 파일 드래그&드롭

▶️ 프리셋 슬롯 사용:
- F1~F5: 해당 슬롯 패턴 로드
- 1~5번 버튼 클릭: 슬롯 패턴 로드
- Shift+F1~F5: 현재 패턴을 슬롯에 저장

🛡️ 데이터 보호
- 중복 이름 저장 시 덮어쓰기 확인
- 최대 저장량 초과 시 가장 오래된 것 자동 삭제
- 잘못된 파일 형식 자동 감지 및 오류 알림

�🚀 5. 고급 기능
================================================================================

📊 시각적 피드백
- 현재 재생 중인 스텝은 흰색 테두리로 표시
- 각 드럼별 고유 색상으로 활성 스텝 표시
- 하단 진행률 바로 전체 진행 상황 확인
- 강박(1, 5, 9, 13박)은 붉은 선으로 표시

🎛️ 실시간 편집
- 재생 중에도 스텝을 클릭하여 즉시 패턴 변경
- 다음 루프부터 변경사항이 적용됨
- BPM 조절도 재생 중 실시간 반영

🎲 스마트 랜덤 패턴
- 음악적으로 자연스러운 패턴 자동 생성
- 각 드럼의 특성을 고려한 확률적 배치
- 4/4 박자 구조를 유지하며 랜덤 생성

📱 반응형 디자인
- 데스크톱, 태블릿, 모바일 모든 기기 지원
- 모바일에서는 세로 스크롤로 12트랙 표시
- 터치 최적화된 인터페이스

================================================================================
💡 5. 팁 & 트릭
================================================================================

🎵 기본 드럼 패턴 만들기
1. Kick(1): 1, 5, 9, 13박에 배치 (4/4 박자 기본)
2. Snare(2): 5, 13박에 배치 (백비트)
3. Hi-hat(3): 모든 8분음표(홀수 스텝)에 배치
4. 다른 드럼들로 변화와 액센트 추가

🎸 장르별 패턴 팁
▶️ 록/팝 패턴:
   - Kick: 1, 5, 9, 13
   - Snare: 5, 13
   - Hi-hat: 1, 3, 5, 7, 9, 11, 13, 15

▶️ 일렉트로닉 패턴:
   - Kick: 1, 5, 9, 13
   - Clap: 5, 13
   - Shaker: 모든 16분음표

▶️ 펑크 패턴:
   - Kick: 1, 7, 10, 13
   - Snare: 5, 13
   - Hi-hat: 촘촘하게 배치

🎹 연주 팁
- 숫자키 1-4로 기본 리듬 연습
- Q, W, E로 필인(fill-in) 연습
- Random 버튼으로 새로운 아이디어 얻기
- 느린 BPM에서 시작해서 점점 빠르게

🔧 성능 최적화
- 브라우저 탭을 활성 상태로 유지
- 다른 오디오 앱 종료 권장
- Chrome, Firefox 최신 버전 사용 권장

🛠️ 6. 문제 해결
================================================================================

❌ 소리가 나지 않는 경우:
✅ 브라우저 볼륨 확인
✅ 시스템 볼륨 확인
✅ 다른 오디오 앱이 사운드를 점유하고 있는지 확인
✅ 페이지 새로고침 후 Play 버튼부터 클릭
✅ 브라우저의 자동재생 차단 해제

❌ 타이밍이 부정확한 경우:
✅ 브라우저 탭을 활성 상태로 유지
✅ 다른 무거운 프로그램 종료
✅ 페이지 새로고침 후 다시 시도

❌ 모바일에서 문제가 있는 경우:
✅ 세로 모드로 사용
✅ 브라우저를 최신 버전으로 업데이트
✅ 다른 앱들을 모두 종료 후 사용

❌ 키보드 단축키가 안 되는 경우:
✅ Beat Drum 페이지에 포커스 맞춤 (페이지 클릭)
✅ Caps Lock이 켜져있지 않은지 확인
✅ 다른 키보드 단축키 프로그램과 충돌 확인
